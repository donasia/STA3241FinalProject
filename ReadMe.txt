For this project I have used two different datasets and combined them to properly analyze them:

The first is from Kaggle. It is called `acnh_villagers_data` in a .csv file format. It includes the popularity of all of the villagers based on the community's rankings.

The second one was found on GitHub. It is called `villagers` in a .csv file format. It includes all of the villagers that are found in the game Animal Crossings: New Horizons (ACNH).

I have included a link for both files if you are interested in the source that I got it from:

`acnh_villager_data` is from https://www.kaggle.com/datasets/ampiiere/acnh-villager-popularity

`villagers` is from https://github.com/ErikaJacobs/AnimalCrossing_PopularityData
